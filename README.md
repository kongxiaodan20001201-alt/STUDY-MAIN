# Java 架构师学习仓库

这是一个使用 AI 辅助学习方法的 Java 架构师学习仓库。作为一名拥有 10 年 Java 开发经验的开发者，这个仓库将帮助你系统性地学习 Java 架构师所需的核心知识和技能。

**学习方法**：使用 Claude Code 作为交互式 Java 架构师导师，采用苏格拉底式教学方法，帮助你深入理解架构设计、系统设计、性能优化等核心概念。

---

## 如何使用

这个仓库使用 Claude Code 作为交互式 Java 架构师导师，它能够：
- 使用苏格拉底式教学方法（先询问你已经知道什么）
- 提供简洁清晰的解释（约 200 字）
- 通过后续问题验证你的理解
- 根据你的回答调整教学风格
- **跟踪每次学习会话，个性化你的学习体验**

## 仓库结构

```
/sessions/                    # 每日学习会话记录
  /2025-XX-XX/               # 每次学习日期一个文件夹
    session-notes.md         # 会话详细记录
  SESSION-TEMPLATE.md        # 会话记录模板

/progress/                    # 学习进度的单一数据源
  java-architect-tracker.md  # 综合跟踪文件，包含：
                             # - 所有 Java 架构师主题映射
                             # - 已掌握的主题
                             # - 识别的知识缺口
                             # - 剩余学习计划

CLAUDE.md                     # AI 导师指导说明（苏格拉底式教学法）
README.md                     # 本文件
```

## 使用方法

### 每日学习会话

1. 在此仓库中打开 Claude Code
2. 自然地询问 Java 架构师相关主题的问题 - 就像和导师对话一样
3. 回答 Claude 提出的理解检查问题
4. 每次会话后，Claude 会自动记录：
   - 你学到了什么
   - 你在哪些方面遇到困难
   - 你掌握了哪些内容
   - 下一步需要复习什么

### 复习会话

当你想要复习时，只需询问 Claude：
- "让我们复习一下我遇到困难的主题"
- "今天我应该关注什么？"
- "测试一下我的薄弱环节"
- "显示我的学习进度"

Claude 会读取你的会话历史，并根据你过去的表现创建个性化的复习内容。

### 跟踪学习进度

查看 `/progress/java-architect-tracker.md` 以了解：
- 整体学习进度
- 哪些知识领域已完成
- 剩余的知识缺口
- 优先学习计划

## 学习理念

**引导式学习方法**：
- 对话式且无评判
- 基于你现有的知识构建
- 在继续前进之前检查理解
- 适应你的学习风格
- 专注于深入理解，而不仅仅是记忆

## 主要特性

**个性化学习**：
- 详细记录的学习会话
- 苏格拉底式教学方法（基于你已经知道的内容构建）
- 根据你的回答调整解释
- 针对薄弱环节的实践问题

**全面跟踪**：
- 每次会话自动记录
- 识别和跟踪知识缺口
- 记录掌握的主题及信心水平
- 根据知识领域权重衡量进度

**基于证据的方法**：
- 所有答案都基于权威来源验证
- 技术问题不猜测
- 复杂规则提供引用
- 专注于理解"为什么"而不仅仅是"是什么"

## 如何开始使用

1. **克隆此仓库**（如果需要）：
   ```bash
   git clone <your-repo-url>
   cd Java-Architect-Study
   ```

2. **清理旧的学习历史**（如果需要重新开始）：
   ```bash
   rm -rf progress/ sessions/
   ```

3. **运行 Claude Code**：
   在 Cursor 中打开此仓库，开始与 Claude 对话

4. **开始学习！** 开始询问 Java 架构师相关的问题，Claude 会：
   - 使用苏格拉底式方法教学
   - 自动创建新的 `progress/` 和 `sessions/` 文件夹
   - 跟踪你的学习旅程
   - 适应你的学习风格

`CLAUDE.md` 文件包含了 Claude 应该如何指导你的所有说明。

## 开始学习

只需开始与 Claude Code 对话，提出你的第一个 Java 架构师问题。Claude 会从那里引导你，同时自动跟踪你的进度。
